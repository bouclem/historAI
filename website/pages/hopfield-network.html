Hopfield Network

📅 1982 👤 John Hopfield 🏷️ Real History

Overview

The Hopfield Network is a recurrent neural network with symmetric weights that stores patterns as attractors — local minima of an energy function. Published by John Hopfield in 1982, this paper is widely credited with reviving interest in neural networks by connecting them to physics (spin glasses, Ising models).

The key insight is that a network with symmetric weights and asynchronous updates will always converge to a stable state — a local minimum of the energy function:

E = -0.5 · Σ_ij W_ij · s_i · s_j

Patterns are stored using Hebbian learning — the weight matrix is the sum of outer products of the stored patterns. When a noisy or partial pattern is presented, the network converges to the nearest stored pattern, acting as an associative memory or content-addressable memory.

For classification, we store one prototype pattern per class and classify by finding which stored pattern the input converges to (minimum Hamming distance after recall).

Benchmark results: The Hopfield Network achieves 100% accuracy on high-dimensional data (10 features) — its sweet spot, where the binary pattern space is rich enough for distinct prototypes. On low-dimensional data (2 features), it hovers around 50–54% because the bipolar binarization loses too much information. The network is extremely fast (one-shot learning, ~1ms) but limited by its capacity: the number of patterns it can reliably store is approximately 0.14 × n_features.
