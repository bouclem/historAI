Adaptive Threshold Neuron

📅 1948 👤 historAI Project 🏷️ Divergence — Neutral

Overview

The Adaptive Threshold Neuron is a divergence architecture — a model that could have existed in 1948, between the McCulloch-Pitts neuron (1943) and the perceptron (1957).

The McCulloch-Pitts neuron had fixed weights and a fixed threshold. The perceptron made weights adaptive but treated the threshold as a bias term. This model takes a different path: it makes the threshold itself a learnable parameter.

The threshold adapts based on prediction errors:

θ(t+1) = θ(t) − r · (d_j − y_j)

When the neuron fires too often (predicts 1 when it should predict 0), the threshold increases — making it harder to fire. When it doesn't fire enough (predicts 0 when it should predict 1), the threshold decreases — making it easier to fire. Weights also adapt simultaneously:

w_i(t+1) = w_i(t) + r · (d_j − y_j) · x_j,i

Whether this approach is better or worse than the perceptron's bias-based threshold depends on testing — convergence speed, stability, and accuracy across different problems.

1943
McCulloch-Pitts Neuron — Fixed Threshold
The original neuron model: fixed weights, fixed threshold. The threshold θ is a constant — the neuron fires if the weighted sum of inputs reaches θ. No learning occurs.
Real History
1948 — DIVERGENCE
Adaptive Threshold Neuron
Instead of keeping the threshold fixed (McCulloch-Pitts) or treating it as a bias weight (perceptron), this model makes the threshold a first-class learnable parameter. The threshold adapts automatically based on prediction errors — increasing when the neuron fires too often, decreasing when it doesn't fire enough.

Perceptron — Adaptive Weights, Bias Threshold
Rosenblatt's perceptron takes a different approach: weights adapt, but the threshold is absorbed into a bias term. The perceptron learning rule updates weights based on errors, but the threshold is just another weight connected to a constant input of 1.
